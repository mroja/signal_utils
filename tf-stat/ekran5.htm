Screen 5

In this step you configure the statistic analysis

Parameters:

  • dt – time span of a resel 

  • statistics &ndash type of test. Options are:

    • Bootstraped T-statistics

    • Permutation test

    • Parametric T-test

    • Welch T-test &ndash parametric T test with Welch correction for unequal variances

    • Parametric Z-test  

  • correction for multiple tests – type of correction for multiple tests to apply. Options are:

    • FDR - False Discoveries Rate

    • BH – Benferoni Holmes

    • NONE – no adjustment of p-level 

  • p level – significance level (for NONE or BH correction) 

  • q (FDR) – acceptable level of false discoveries (fraction of resels falsely indicated as significant among all significant resels) for FDR correction 

  • Number of permutations – number of permutations/bootstraps in resampling based tests
