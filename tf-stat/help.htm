Event related time-frequency maps

 

 

 

General informations:

This software, the TFStats package for calculating significant ERD/ERS in EEG data and similar paradigms, is an implementation of methodology for investigation of Event related power changes in bio-electrical signals in time-frequency plane. The mtehodology is described in details in papers:

1) Computationally efficient approaches to calculating significant ERD/ERS changes in the time-frequency plane J. ¯ygierewicz, P.J. Durka, H. Klekowicz, N. Crone, P. Franaszczuk, Journal of Neuroscience Methods Vol 145/1-2, pp. 267-276, 2005

2) On the statistical significance of event-related EEG desynchronization and synchronization in the time-frequency plane P.J. Durka, J. ¯ygierewicz, H. Klekowicz, J. Ginter, K.J. Blinowska IEEE Transactions on Biomedical Engineering Vol. 51, Issue 7, pp. 1167-1175, July 2004

Running the software:

To run the software, execute script TF_GUI from Matlab prompt. The software was tested in Matlab ver. 7 (should also run in ver. 6).

To generate t-f maps you need to configure parameters for the procedures.

You can load a predefined configuration from a file and then adjust it or start from scratch and complete the configuration in five steps.

The steps group parameters in the following categories:

  • File Configuration

  • Signal Parameters

  • Maps Configuration

  • Axes Parameters

  • Statistics

To advance to a next step press button "next" to return to a previous step press "back". At certain steps some of the options may be grayed. That means this options is inactive (It is irrelevant in the present context).

On the last (5-th) step there is a "start" button, which starts the computations.

The configuration can be saved for future use in a file (this can be done in step 5-th).
